## Project Description
Implementation of a **Huffman Encoder** in C programming language.

Huffman coding is a lossless data compression algorithm that uses variable-length codes to represent symbols. More frequent characters are assigned shorter codes, resulting in efficient compression.

### Algorithm Overview
1. Calculate frequency of each character in the input
2. Build a priority queue (min-heap) based on frequencies
3. Build a Huffman Tree from the priority queue
4. Generate Huffman codes by traversing the tree
5. Encode the input using the generated codes

### Features
- Frequency analysis of input text
- Binary tree construction (Huffman Tree)
- Variable-length encoding
- File compression and decompression
